## SIG — 신호등 ("지금 잡을 딜이 있는가")

### SIG-01 신호 색 (자격 = signalSet, 색 판정 = priceLast)
| 색 | 의미 |
|---|---|
| 🟢 | P25 이하 활성 딜 존재 |
| 🟡 | 기준가 이하 활성 딜 존재 |
| 🔴 | 없음("딜 없음", 갭 병기 — 캐시 히트 시, 갭 ≤ 0이면 강조) |
| ⚪ | SPARSE·NONE(⚪+딜 = SPARSE 전용, 자기 제외 사례 ≥ 1일 때만 비교 문구) |
- m = 0 → 🟢 불가 + "굿딜라인 미확립".
- **표시 전용 — 알림 트리거 금지, 신호 변화 알림 금지**(B-5 비대칭: 상시 표시는 단정, signalSet만).

### SIG-02 신선도 3단 (관측시계 기반, `docs/03` 3-2)
| 단계 | 처리 |
|---|---|
| 확신 | 상한(Q-24, 가안 48h) 내 |
| 약화 | 상한~자격 한도, 색 + 주석 |
| 자격 상실 | 신호 제외, "최근 딜: N일 전 X만원(종료 미확인)" 정보 강등 |
- 백필 딜은 postedAt 기산으로 자연히 한도 밖.

### SIG-03 표면 × 상태
- 목록: 색 점 + 관찰 배지 + 📌점 자리 분리 병존. API 무트리거(🔴 색은 DB, 갭%만 캐시).
- 상세: 전개 + "검토 대기 N건" 통합 표시.
- 기능4(구매 비교): 일반 신호(관찰 중에도).
- /status: 목록 셀 규칙.
- 다이제스트·보관함: 행 포함.
- SPLIT = 값 집계 최고 신호.
- 관찰 문맥 시 신호 자리 전환(잣대 paidPrice — PUR 알림 술어 공유, PUR-03).

### SIG 열린 것

## CAD — 딜 주기 ("기다리면 또 오는가")

### CAD-01 산출 (집합 = occurrenceSet, 축 = firstSeen)
- `eventCount` · 간격 median: 창 = P ∩ [observedFrom, now](확장 없음), 가드 n ≥ K_display(간격 표본 = eventCount − 1).
- 경과일 = P 무관 · 관측 범위 내 최신 firstSeen · 조회 시점 파생(캐시 제외).
- **"다음 딜 예상일" 출력 금지**(예측 금지). 라이브 = 현재 설정·현재 지식.

### CAD-02 표기
- 자기 분모 명시. 가드 미달 + 경과일만이면 "주기 판단 불가(발생 N건)" 병기.
- caveat 클러스터: 시즌성 / 구간 절단 무보정 / 재풀림(동일 이벤트 내 부활만 미포함 · 윈도우 밖 재등록은 신규 집계) / 지속형 딜 / 수집 공백.

### CAD-03 표면
- 상세 O / 기능4 O(신호·갭과 한 블록) / 목록 X / status X / 다이제스트 X.
- SPLIT = 값별 + 미상 공시.

### CAD 열린 것
- 없음(P 기본값은 Q-26, `docs/03` 손잡이 목록).

- SIG 색 판정: signalSet 자격 × priceLast × (m=0, SPARSE 자기제외, 갭≤0) 매트릭스. 신선도 3단 경계(48h·7일). 표시 전용(알림 트리거 없음) 보장.
