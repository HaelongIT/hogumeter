# 03. 딜 집합 명명 & 시간·관측 좌표계 (2차 기반)

> 출처: `working-area/2nd-plan-intake.md` B-1·B-2·B-6. 2차 기능(PUR/SIG/CAD/DIGEST/WATCH/PRI)이 공유하는 **기반 정의**. 여기서 정의한 집합·시각·as-of 규약을 각 기능 문서가 참조한다.
> ⚠️ 일부 조항은 확정본 충돌 후보에 걸려 있음(`working-area/decisions-needed.md` DN-C2·C3·C5) — 해당 부분은 **[잠정]**으로 표기, 승인 시 확정.

## 3-1. 세 집합 명명 계층 (pricingSet / occurrenceSet / signalSet)

**문제**: 1차의 단일 어휘 "유효 딜"이 세 질문(값이 얼마인가 / 언제 얼마나 자주인가 / 지금 잡을 게 있는가)을 겸해 오독·오구현을 유발. → 질문별로 집합을 분리 명명한다.

| 집합 | 질문 | 소비처 | 자격 술어 |
|---|---|---|---|
| **pricingSet** | 값의 통계 | 기준가·P25·성적표 percentile | CONFIRMED + 키워드 무히트 + 선택 축값/basis 일치 + 이상치 미해당(**승격된 LOWER는 포함**) + 배치유보 제외 |
| **occurrenceSet** | 시간의 통계 | CAD 딜 주기 | 정체성 축만 — CONFIRMED + 키워드 무히트 + 선택 축값 일치 + **UPPER 제외**(가격으로 감지된 정체성 실패 프록시) + **LOWER는 기각만 제외** + **배치유보 포함** |
| **signalSet** | 지금의 단정 | 신호등·상시 표시 | CONFIRMED + 키워드 무히트 + **무경고(⚠️ 라벨 제외 [잠정 DN-C5])** + 비이상치 + ENDED 아님 + 신선도 자격 + 선택 축값. **트리거(목표가/paidPrice) 무관** |

**규약**
- 집합 × 창(window)은 **독립 차원**. 집합별 tier(SUFFICIENT/SPARSE/NONE) 독립 판정.
- **모든 통계는 자기 집합·자기 분모를 명시**한다(정직성). "n건(교차 m건)"은 pricingSet 분모.
- 상시 표시(단정)는 signalSet만, 알림(재현율)은 트리거 계열 — B-5 비대칭 원칙.

### 동사 재조준 표 (1차 문장의 집합 목적어 매핑)
1차 문서의 "제외/포함/배제"가 **어느 집합을 대상으로 하는지** 명시. 신규 조항 아님 — 기존 문장 해석 고정.

| 1차 표현 | 재조준 |
|---|---|
| "이상치는 기준가에서 제외" | UPPER→pricingSet 제외·occurrenceSet 제외 / LOWER 미확정→pricingSet 제외·occurrenceSet 포함 |
| "단일 사이트 딜 포함(가중분리)" | pricingSet median 포함, P25·역대최저는 교차검증만 |
| "미상 버킷은 계산 제외" | 세 집합 모두 제외(사람 분류 시 편입) |
| "제외 키워드 배제/⚠️라벨" | 세 집합 모두 통계 제외, ⚠️는 표시만 [잠정 DN-C5] |

## 3-2. 시간·관측 좌표계

**문제**: "부재 = 정보(딜이 없었다) vs 무지(관측을 못 했다)"의 구분 축이 1차에 없어 시간 통계·신선도가 관측 공백을 딜 부재/노화로 오독.

### firstSeen (발생 시각·불변) [잠정 DN-C2]
- = **발생 시각**: 라이브도 글 postedAt 우선, 파싱 실패 시 첫 관측 폴백 / 백필 = postedAt(날짜만이면 23:59 KST).
- 생성 시 **1회 확정·불변** — 재분류·후행 postedAt 확보로도 소급 변경 없음.
- `capturedAt`(수집 시각) 별도 보존(항상).
- firstSeen 독자 소비 목록: 기준가 기간 필터 / CAD 산입·as-of / PUR 산입 / 병합 윈도우 / lastEvidenceAt 성분.
- ⚠️ 라이브 postedAt 우선은 **M0 실측 후 확정**(`docs/91` Q-23).

### lastEvidenceAt (살아있음의 적극 증거)
- = max(최신 병합 글 firstSeen, 마지막 PRICE_CHANGED). **스칼라**.
- 단순 생존 관측(변화 없는 재확인)은 불포함 — "적극 증거"만.

### observedFrom (variant별 관측 시작)
- = min(백필 성공 사이트 도달 시점) 또는 등록 시각.
- **유효 창 규칙**: 모든 시간 통계 = 대상 창 ∩ [observedFrom, now/발급]. 창이 짧으면 "관측 범위 N개월" 표기.
- K_fill 자동확장 상한도 observedFrom로 제약 [잠정 DN-C3].

### 관측시계 (staleness)
- 신선도 staleness = (소스 사이트 마지막 성공 폴링) − lastEvidenceAt. 수집 공백 시 **정지**(무지를 부재로 오독 방지).
- 시간 통계 내부의 관측 공백 = caveat 수용 / 실시간 판정(신호)은 시계 정지(처리 차이 병기).

## 3-3. as-of 재구성 소비처 규약 (B-6)

기준가 엔진의 **as-of 재구성**(순수 함수)을 여러 소비처가 다른 시점·축값으로 호출한다. 규약을 고정해 화면 간 숫자 차이가 무설명이 되지 않게.

| 소비처 | 시점 | 축값/basis |
|---|---|---|
| 스냅샷(PUR) | purchasedAt의 함수 + 기록 시점 설정 | 동결 basis |
| 성적표(PUR) | 발급 시점 설정·지식 | 동결 basis 재현 |
| 회고(WATCH) | 조회 시점 | 딜 실제 축값 |
| 다이제스트 | 발송 시점 | 딜 실제 축값 |

- **일반화**: 라이브 뷰 = 실제 축값 / 확정물 = 동결 basis. "확정물은 인용, 라이브만 계산" / "복귀는 신인".
- "유령 분포 금지"는 **라이브 기본 잣대 한정** — 재현 계산(as-of)은 항상 허용.

## 3-4. 미산입 공시 규격 (B-6)
- 통합 규격: **[창 명시 → 통계 → "미산입 N건(사유: 후보·의심·미상·묶음의심)"]**.
- 공시 창 = 대상 통계와 동일 창. **기각(REJECTED)은 공시 비대상**(이력 열람만).

## 3-5. 손잡이(설정값) 독자 목록 (B-6)
소비처가 늘면 이 목록을 갱신할 의무(관례).

| 손잡이 | 소비처 |
|---|---|
| `K_display` | 기준가 라벨 / K_fill 성분 / CAD 가드 |
| `P`(분석 기간) | 기준가 + CAD (공용, 의미 확장) |
| 우선순위 순번 | 목록 정렬 1곳 (PRI ②축소 채택, DN-P — 알림 병기 드랍) |

## 테스트 포인트
- 집합 자격 술어: 각 집합 × (CONFIRMED/CANDIDATE, 키워드 히트, UPPER/LOWER 미확정/승격/기각, 배치유보, ⚠️라벨, 신선도 자격) 파라미터라이즈드 — 동일 딜이 집합마다 다르게 분류되는지.
- 시간 좌표: firstSeen 불변(재분류·후행 postedAt 후에도), 유효 창 ∩ observedFrom, 관측시계 정지(수집 공백).
- as-of: 소비처 4종 시점·축값 규약이 같은 함수에서 다른 결과를 내는지(동결 vs 라이브).
