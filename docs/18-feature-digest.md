# 18. 기능(2차)5 — 주간 요약 다이제스트 (DIGEST)

> 출처: `working-area/2nd-plan-intake.md` B-11. 실시간 알림(지금)의 보완재(흐름) + 놓침 안전망. 안전망의 질문은 "발생"이 아니라 **"가시화"**. 신규 수집 0.
> 마일스톤: DN-M(권고 M5, PUR·SIG 이후). 알림 계열 연동은 `docs/12` B-5.

## DIG-01 제3계열 (정기 리포트)
- quiet 존중·관통 없음, **발송 단위 원칙 관할 밖**(스케줄 구동 별축).
- 기본 일요일 20시 KST(사용자 설정). quiet 겹침 시 종료 후 발송.

## DIG-02 저장물 (variant당)
- (신호값, 기준 시각) + 3성분(색 / 관찰 문맥 / basis 모드 — 뒤 2개 = 전환 억제 신호).
- SPLIT은 목록 셀 집계로 variant당 1벌.
- **발송 성공 후 갱신**(REL-03 원자성, `docs/20`). 분할 발송(절단 금지·연번·전 분할 성공 시 갱신·quiet 판정은 사건 시작 1회).
- 신호 이력 테이블은 두지 않음(직전 발송 1벌 저장으로 전환 계산 — D 폐기안).

## DIG-03 창 & 시점 규약
- 창 = **[max(직전 성공 발송, 활성 시각), 이번 발송) 반개구간 ∩ 관측 가동 구간**.
- "복귀는 신인": 기능 재활성화·ARCHIVED 복귀 = 기준점 무효화.
- 시점 규약 = "발송 시점 현재 지식"(헤더 1회 명시). 확정물은 인용만(성적표 수치 재서술 금지).
- **플로우 귀속 = 가시화 시각**(occurrenceSet 자격을 최초/재획득한 시각 — 늦확정·백필·재진입 커버). "발생은 N일 전" 병기는 창 경계 초과 시만. 자격 상실 방향은 비보고(사유 명기).

## DIG-05 규칙
- 플로우/스톡 분류·문구("지난 N일" / "현재 기준"). 정렬 = 정보 가치순 + 무변동 합산.
- 배치 중 variant "수치 잠정" 병기. 관측 공백 헤더 2단(전체 / 일부).
- **중고 축 기각**(사유: 1점물·기준가 무합성·후속 알림이 실시간 담당). **CAD 미포함**.
